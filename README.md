# SOC Analyst I ‚Äî From Cradle to Grave

A practical, intensive 8-module cybersecurity course designed to transform an aspiring professional into a job-ready Security Operations Center (SOC) Analyst I.

## Course Overview

This course focuses on real-world incident response, detection engineering, and threat hunting across modern hybrid environments (Windows, Linux, and AWS Cloud). The curriculum is structured around the 8 most common and critical attack types a Tier 1 SOC analyst will face, providing a hands-on, portfolio-building experience.

## Learning Objectives

Upon completion of this course, you will be able to:

- Handle Tier 1 SOC tickets end-to-end
- Perform triage on phishing, malware, DDoS, data exfiltration, and unauthorized access
- Write clean, concise SOC reports and ticket updates
- Perform NetFlow and PCAP traffic analysis
- Investigate host-based logs on Linux and Windows
- Run memory forensics
- Execute OSINT enrichment
- Create detection rules using Suricata, YARA, Sigma
- Operate in hybrid environments (Windows + Linux + AWS CloudTrail + VPC Flow Logs)
- Respond to live, simulated attack campaigns
- Pass SOC Tier 1 and Tier 2 interviews with confidence
- Create a public cybersecurity portfolio with real projects

## Prerequisites

- Basic understanding of networking (TCP/IP, OSI model)
- Familiarity with Linux command line
- Basic knowledge of cybersecurity concepts
- Access to a Linux system or VM for labs

## Tools Required

All labs use open-source tools:
- Wireshark (network analysis)
- Volatility Framework (memory forensics)
- Suricata (IDS/IPS)
- YARA (malware detection)
- Linux command-line tools
- AWS CLI (for cloud security modules)

## Study Schedule

The course is designed for learners working 10-hour shifts who can study 1-2 hours per night. The recommended schedule is:

- **90-Day Plan**: Complete all 8 modules over 12 weeks
- **Daily**: 1-2 hour study sessions
- **Weekly**: Complete 1-2 module sessions

See `docs/roadmaps/90-day-study-schedule.md` for detailed planning.

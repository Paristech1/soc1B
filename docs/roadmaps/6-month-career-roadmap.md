# 6-Month SOC Career Roadmap

**Tier 1 → Tier 2 → Specialist**

This roadmap outlines your progression from SOC Analyst I to specialized roles over 6 months.

---

## Months 1-2: SOC Analyst I (Foundation)

### Goals
- Complete the "SOC Analyst I — From Cradle to Grave" course
- Master Tier 1 skills: phishing triage, basic malware analysis, incident ticket creation
- Build a portfolio of 8-10 projects demonstrating core skills

### Skills to Develop
- **Phishing Triage:** Email header analysis, OSINT, PCAP analysis
- **Malware Analysis:** Static analysis, basic dynamic analysis
- **Incident Response:** NIST 800-61 framework, ticket creation
- **Network Analysis:** Wireshark, PCAP analysis
- **Host Log Analysis:** Linux auth.log, Windows Event Log
- **Ticketing:** Incident ticket creation and management

### Portfolio Projects (8-10 projects)
1. Phishing Triage Playbook
2. Unauthorized Access IR Report
3. Data Exfil Detection Blueprint
4. DDoS Triage Workflow
5. Malware Triage Summary Report
6. Detection Rule Pack v1
7. Full Incident Report with Timeline
8. 10-Minute Incident Summary Challenge
9. [Additional project of your choice]
10. [Additional project of your choice]

### Certifications to Pursue
- **CompTIA Security+** (foundational certification)
- Start studying for **CEH** or **GSEC**

### Key Milestones
- ✅ Complete all 8 course modules
- ✅ Build portfolio with 8-10 projects
- ✅ Pass Security+ exam
- ✅ Create LinkedIn profile showcasing portfolio
- ✅ Apply for SOC Tier 1 positions

---

## Months 3-4: SOC Analyst I (Advanced) → Tier 2 Preparation

### Goals
- Deepen skills in detection engineering, threat hunting, and cloud security
- Begin contributing to detection rule development
- Study for advanced certifications
- Start applying for Tier 2 positions

### Skills to Develop
- **Detection Engineering:** Advanced Suricata, YARA, Sigma rules
- **Threat Hunting:** Proactive threat detection, hypothesis-driven investigations
- **Cloud Security:** AWS CloudTrail, GuardDuty, VPC Flow Logs analysis
- **Linux Forensics:** Advanced log analysis, disk forensics
- **Automation:** Bash scripting, Python for SOC automation
- **Threat Intelligence:** IOC management, threat intel integration

### Portfolio Projects (5-7 projects)
1. Advanced Detection Rule Pack (10+ rules)
2. Threat Hunting Report
3. Cloud Security Assessment
4. Linux Forensics Analysis Report
5. SOC Automation Scripts
6. Threat Intelligence Integration Project
7. [Additional project of your choice]

### Certifications to Pursue
- **Certified Ethical Hacker (CEH)** or **GIAC Security Essentials (GSEC)**
- **AWS Certified Security - Specialty** (if focusing on cloud security)

### Key Milestones
- ✅ Complete advanced detection engineering projects
- ✅ Pass CEH or GSEC exam
- ✅ Contribute detection rules to open-source projects
- ✅ Build threat hunting capabilities
- ✅ Apply for SOC Tier 2 positions

---

## Months 5-6: SOC Analyst II / Specialist Role

### Goals
- Transition to Tier 2 role or specialized role (Cloud Security Analyst, Threat Hunter)
- Complete advanced certifications
- Build expertise in chosen specialization
- Mentor junior analysts

### Skills to Develop
- **Advanced Threat Hunting:** Hypothesis-driven investigations, threat actor tracking
- **Cloud Architecture:** AWS security architecture, cloud security best practices
- **Security Automation:** Advanced scripting, SIEM integration, orchestration
- **AI-Augmented SOC:** Leveraging AI/ML for security automation
- **Incident Response Leadership:** Leading incident response efforts
- **Mentoring:** Training and mentoring junior analysts

### Portfolio Projects (3-5 projects)
1. Advanced Incident Report (multi-stage APT attack)
2. Security Automation Framework
3. Cloud Security Architecture Design
4. Threat Actor Profile and Tracking Report
5. [Specialization-specific project]

### Certifications to Pursue
- **AWS Certified Security - Specialty** (for cloud security path)
- **GIAC Certified Incident Handler (GCIH)** (for IR specialization)
- **GIAC Certified Threat Intelligence Analyst (GCTI)** (for threat hunting)

### Key Milestones
- ✅ Secure SOC Tier 2 or Specialist position
- ✅ Complete advanced certifications
- ✅ Build strong portfolio demonstrating expertise
- ✅ Establish yourself as a subject matter expert
- ✅ Begin mentoring junior analysts

---

## Specialization Paths

### Cloud Security Analyst Path
- **Focus:** AWS, Azure, GCP security
- **Skills:** CloudTrail analysis, IAM security, cloud architecture
- **Certifications:** AWS Security Specialty, Azure Security Engineer
- **Portfolio:** Cloud security assessments, cloud incident response reports

### Threat Hunter Path
- **Focus:** Proactive threat detection, threat actor tracking
- **Skills:** Hypothesis-driven investigations, threat intelligence, behavioral analysis
- **Certifications:** GCTI, SANS FOR508
- **Portfolio:** Threat hunting reports, threat actor profiles

### Detection Engineer Path
- **Focus:** Detection rule development, SIEM management
- **Skills:** Advanced rule writing, SIEM architecture, threat detection
- **Certifications:** Vendor-specific SIEM certifications
- **Portfolio:** Detection rule packs, SIEM optimization reports

## Portfolio Building Strategy

### Month 1-2: Foundation Projects
- Focus on demonstrating core SOC skills
- Use course labs as starting points
- Document everything clearly
- Share on LinkedIn and GitHub

### Month 3-4: Advanced Projects
- Build on foundation projects
- Add complexity and real-world scenarios
- Contribute to open-source projects
- Write blog posts about your projects

### Month 5-6: Specialization Projects
- Focus on your chosen specialization
- Demonstrate advanced expertise
- Create tools or frameworks
- Present at meetups or conferences (if possible)

## Networking & Community Engagement

### Months 1-2
- Join cybersecurity communities (Reddit, Discord, Slack)
- Follow security professionals on LinkedIn/Twitter
- Attend local security meetups (virtual or in-person)
- Engage with course community

### Months 3-4
- Contribute to open-source security projects
- Write blog posts about your learning journey
- Share portfolio projects on LinkedIn
- Connect with SOC analysts and managers

### Months 5-6
- Present at security meetups or conferences
- Mentor junior analysts

- Establish yourself as a thought leader in your specialization
